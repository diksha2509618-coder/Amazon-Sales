# Amazon-Sales

## 📊 Project Overview
This project is an interactive dashboard created using Google Data Studio to visualize and analyze data in a clear and meaningful way. The dashboard provides a consolidated view of key performance indicators (KPIs), trends, and insights through charts, tables, and filters, enabling better data-driven decision-making.

## 🛠 Tools Used
- Looker Studio
- Microsoft Excel

## 📁 Dataset
The dataset used for this project is included in this repository (Excel file).

## 📈 Key Insights
- Sales increased in the last quarter
- Electronics category performed best
- Online payments were most preferred
